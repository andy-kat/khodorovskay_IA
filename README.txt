Простая программа для хранения и просмотра цитат.

Что делает
Показывает список известных цитат. Позволяет добавлять новые, удалять ненужные и быстро находить цитаты по автору или теме.

Как пользоваться
Запуск
Установите Python.
Запустите файл скрипта.
Управление списком
Сгенерировать цитату: добавляет в список новую случайную цитату.
Удалить выбранную: удаляет из списка цитату, на которой вы щёлкнули.
Поиск (фильтрация)
Введите автора или тему в верхние поля.
Нажмите «Фильтровать» — в списке останутся только подходящие цитаты.
Чтобы увидеть весь список заново, нажмите «Показать все».
